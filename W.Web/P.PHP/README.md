# P.PHP

## :a: Install with Package Managers

:pushpin: Windows

```
PS > choco install php
```

:pushpin: Apple

```
% brew install ph
```

## :b: Testing

```
% php --version
```

Where is the `.ini` file

```
% php --ini

Scan for additional .ini files in: (none)
Additional .ini files parsed:      (none)
```
