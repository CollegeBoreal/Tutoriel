## Prérequis

* Installer Docker Machine - avec un Package Manager

:pushpin: Windows

```
PS > choco install docker-machine
```


* [Installer Docker Machine - Manuellement](https://docs.docker.com/v17.09/machine/install-machine/)   

:pushpin: MacOS

```
PS > brew install docker-machine
```
