# :m: Python

## :one: Installation de la machine virtuelle Python

:pushpin: Par un Package Manager

:bulb: Windows

```
PS > choco install python3 --pre
```

:apple: MacOS 

```
$ brew install python
$ echo 'export PATH="/usr/local/opt/python/libexec/bin:$PATH"' >> ~/.bashrc
```

:pushpin: Manuellement
